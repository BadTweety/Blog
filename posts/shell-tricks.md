A few things my fingers refuse to memorize.

## 1. Re-run the last command with sudo

```bash
sudo !!
```

## 2. Jump back to the previous directory

```bash
cd -
```

Bounces between two folders — surprisingly useful when comparing configs.

## 3. Fix a typo in the last command

```bash
^typo^fixed
```

Runs the previous command with `typo` replaced by `fixed`. Ancient, obscure, delightful.
